Key Features:

  • Landscape Only: A full-screen "Rotate Device" prompt will block portrait mode on mobile.
  • Responsive HUD: On desktop, it stays as a top/bottom bar. On mobile landscape, it shifts to the sides.
  • Fill-to-Fit: The maze will automatically scale to the largest possible size that fits between the HUD bars.
